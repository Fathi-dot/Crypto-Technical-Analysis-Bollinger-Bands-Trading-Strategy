This project was completed as part of a FinTech exam and focuses on technical analysis, time-series data, and strategy evaluation in cryptocurrency markets.

The tasks include:

Acquiring historical price data for multiple cryptocurrencies

Computing technical indicators: Simple Moving Averages (SMA), Exponential Moving Averages (EMA), and Bollinger Bands

Implementing a Bollinger Bands trading strategy

Testing the performance of the strategy across many randomly selected cryptocurrencies

Evaluating results using financial performance metrics

Discussing whether such trading rules are viable in real crypto markets
